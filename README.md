# Gym Website 

This is a responsive and user-friendly **Gym Website** created using **HTML, CSS, and JavaScript**.  
The project is designed to represent a modern fitness center with clean UI and smooth user experience.

---

## 🔹 Features

- Responsive design for desktop and mobile
- Modern and clean UI layout
- Navigation bar with smooth scrolling
- Sections for:
  - Home
  - About Gym
  - Services / Plans
  - Contact
- Simple and beginner-friendly code structure

---

## 🔹 Technologies Used

- HTML5
- CSS3
- JavaScript

---

## 🔹 Purpose of Project

This project was built to:
- Practice front-end web development skills
- Understand website structure and responsive design
- Create a real-world project for portfolio and resume

---

## 🔹 How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Run `index.html` in any modern web browser
